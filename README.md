## v3.1 Modern UI Updates

- Professional modern clinic layout
- Same MasterCare cream, gold, and brown brand color
- Khmer font support with Kantumruy Pro
- Glass-style sticky navigation
- Rounded premium hero section
- Modern service, promotion, result, and product cards
- Grid and list layout toggle icon
- Smooth scroll reveal animation
- Active menu highlight while scrolling
- Better mobile navigation and spacing
- Cleaner responsive UX inspired by modern portfolio-style layouts
